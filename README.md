# 📁 Portfolio

React와 TypeScript를 사용하여 개발되었으며, Material-UI를 활용한 모던한 디자인을 구현했습니다.

## 🛠️ 기술 스택
- **패키지 매니저**: Yarn
- **빌드 도구**: Vite
- **프레임워크**: React
- **언어**: TypeScript
- **스타일링**: Material-UI (MUI)

## 🚀 시작하기
### 1. 의존성 설치
```
yarn install
```

### 2. 빌드
```
yarn build
```
